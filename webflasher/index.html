Cómo usar: conectá el Arduino Nano por USB, hacé clic en Conectar y grabar y elegí el puerto (suele aparecer como CH340 o USB-Serial). No abras el Monitor Serie ni el IDE al mismo tiempo.
Si dice que el bootloader no responde, marcá Invertir reset y reintentá.
Requiere Chrome, Edge u Opera de escritorio (la Web Serial API no existe en Firefox/Safari ni móvil).
